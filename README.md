## 📌 주요 기능
<div align="center">
    <img width="472" alt="스크린샷 2024-09-01 오후 10 57 39" src="https://github.com/user-attachments/assets/49cb8ab4-d296-4129-8e98-67030fd02158">

🎊 **모임 기능** 🎊  
- 모임 게시판 CRUD
- 검색 / 필터링
- 모임 기능 안내 팝업 (메인 페이지)

💬 **1:N 채팅** 💬  
- 모임 별 채팅방 신설
- 채팅방 나가기 기능

⭐️ **학점(리뷰) 기능** ⭐️  
- 모임 후 모임에 대한 학점(리뷰) 기록
- 학점 통계(미완성)

🔑 **로그인/회원가입 기능** 🔑  
- 사용자 로그인 및 회원가입
- 사용자 세션 관리
